Siti Nur Hanifah

Saya merupakan Siswa jurusan Pengembangan Perangkat Lunak dan Gim Di SMKN 4 Kota Bogor. Saya memiliki minat besar dalam menulis dan sedang belajar berbagai bahasa pemrograman seperti HTML, CSS, PHP, dan JavaScript serta framework Laravel di sekolah. Saya berpengalaman menyelesaikan berbagai project selama bersekolah dengan bantuan para guru. Proyek ini menggunakan HTML, CSS, PHP, dan JavaScript, serta diintegrasikan dengan framework Laravel yang mana membuat saya tertarik dalam bidang IT.

Sejak Di Sekolah Dasar saya senang mengikuti organisasi mulai dari Bela diri sampai Organisasi Relawan
Saat ini saya aktif di organisasi Palang Merah Remaja. Saya sering mengikuti perlombaan PMR tingkat wira yang diadakan di sekolah lain.
Bergabung dalam sebuah organisasi membuat pengalaman saya lebih banyak,dan pengetahuanpun bertambah, melatih jiwa kepemimpinan,
kerjasama tim, meningkatkan kemampuan berkomunikasi dan menjadi kuat dalam menghadapi tekanan.
